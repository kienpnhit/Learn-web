Giới thiệu

HTML là ngôn ngữ để xây dựng cấu trúc trang web.

Bài viết: Tại sao học HTML?

Học HTML giúp bạn tạo nội dung web, kết hợp với CSS & JS để làm website hoàn chỉnh.

Các bước học HTML cơ bản:

  1. Học các thẻ HTML cơ bản (p, h1-h6, a, img…)
  2. Học danh sách: ul, ol, li
  3. Học bảng: table, tr, td
  4. Làm quen với các thẻ semantic (header, nav, main, section, article, footer…)

Thẻ span, hr, br

Tôi thích HTML, CSS và JavaScript.

Dùng thẻ br để xuống dòng:
Đây là dòng thứ 2
Đây là dòng thứ 3

Ví dụ về các thẻ định dạng văn bản

Bold text → Chữ in đậm (trang trí).

Important text → Chữ quan trọng (nhấn mạnh ý nghĩa).

Italic text → Chữ in nghiêng (trang trí).

Emphasized text → Chữ nhấn mạnh (ngữ nghĩa).

Marked text → Chữ được đánh dấu (highlight).

Smaller text → Chữ nhỏ hơn.

Deleted text → Chữ bị gạch bỏ.

Inserted text → Chữ được chèn thêm.

H2O → Chữ dưới dòng (subscript).

x2 + y2 → Chữ trên dòng (superscript).
